1. documents.html (文书列表页)

作用：展示所有文书的列表。用户可以查看所有已登记的文书信息。

功能：

在页面中展示文书名称、朝代、简介等信息。

每个文书名称是一个链接，点击后跳转到该文书的详细页面 (document_detail)。

每个文书下方有一个按钮，允许用户查看该文书下的所有资源分类（跳转到 document_resources 页面）。

2. document_detail.html (文书详情页)

作用：展示单个文书的详细信息。

功能：

显示文书的朝代、简介等

展示文书的相关资源（跳转到 document_resources 页面）以及资源类型。

允许用户查看该文书的具体资源内容，点击每个资源将跳转到 document_resources 页面。

3. document_resources.html (文书资源分类页)

作用：展示文书下的所有资源，并按资源类型进行分类。

功能：

页面展示文书的资源类型（例如：原文、简体版、白话文翻译等），并列出每种资源下的具体内容。

用户可以点击每个资源名称，跳转到 resource_detail 页面查看该资源的详细信息。
